# app

## Build Setup

``` bash
# Install dependencies
npm install

# Start developing with REPL
npm run dev

# Start production
npm start

# Run unit tests
npm test

# Run continuous test mode
npm run ci
```

## Run in Docker

```bash
$ docker-compose up -d --build
```
